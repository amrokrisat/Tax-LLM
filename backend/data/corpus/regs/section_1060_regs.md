Regulation 1.1060-1 supplies the operational allocation rules that make or break the value of an asset path. For this wedge, it matters because the buyer's claimed step-up benefit is only as good as the residual-method allocation that can be supported and documented. The regulation also sharpens seller-side cost analysis because allocation disputes can shift value across ordinary-income, capital-gain, and amortizable buckets. A stock-versus-asset recommendation is incomplete if the modeled step-up ignores these allocation mechanics.
