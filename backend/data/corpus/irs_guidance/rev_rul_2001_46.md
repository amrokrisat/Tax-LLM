Revenue Ruling 2001-46 remains useful as a narrow IRS source on integrated acquisition steps in reorganization analysis. It should not outrank the Code or regulations, but it helps when the structure comparison turns on whether linked merger steps can still be treated as part of one acquisitive reorganization plan.
